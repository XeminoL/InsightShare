#### Overview

The storage layer keeps user files in a **private Amazon S3 bucket** (not public). Uploads and downloads go through **presigned URLs** generated by the back-end, so the browser transfers files **directly** to S3 without routing through Lambda while the bucket stays non-public.

Main steps in this section:

+ Create the S3 bucket for files (region `ap-southeast-1`).
+ Block public access and enable encryption.
+ Configure **CORS** so the browser can call the bucket.
+ Generate a **presigned URL** with **boto3** and test upload/download.
