Kỳ thực tập tại **Công ty TNHH Amazon Web Services Việt Nam** (chương trình First Cloud AI Journey) diễn ra từ **01/06/2026** đến **15/08/2026**.

Sản phẩm chính là **InsightShare**, ứng dụng web upload và chia sẻ ảnh/tài liệu theo hướng serverless trên AWS. Quá trình làm việc trực tiếp với S3, Lambda, API Gateway, DynamoDB, CloudFront, IAM, CloudWatch; ngôn ngữ Python (boto3); tài liệu kỹ thuật viết song ngữ.

Bảng dưới tự đánh giá theo tám tiêu chí, mỗi tiêu chí một mức: Tốt / Khá / Trung bình.

| STT | Tiêu chí | Tốt | Khá | Trung bình | Nhận xét |
| --- | --- | --- | --- | --- | --- |
| 1 | **Kiến thức và kỹ năng chuyên môn** | x | | | Áp dụng được các dịch vụ AWS cốt lõi vào một dự án end-to-end. |
| 2 | **Khả năng học hỏi** | x | | | Tự học các dịch vụ mới (Lambda, presigned URL, CloudFront) qua tài liệu và lab. |
| 3 | **Tính chủ động** | | x | | Chủ động nhận việc và tự tìm hiểu; đôi lúc cần thêm định hướng. |
| 4 | **Kỷ luật** | | | x | Từng bị nhắc về trang phục. |
| 5 | **Giao tiếp** | | x | | Báo cáo công việc bằng văn bản rõ; trao đổi trực tiếp còn dè dặt. |
| 6 | **Hợp tác nhóm** | | x | | Hòa nhập, hỗ trợ khi được hỏi; ít chủ động mở trao đổi nhóm. |
| 7 | **Tư duy giải quyết vấn đề** | | x | | Tự gỡ các lỗi tích hợp (CORS, IAM, cold start) chủ yếu bằng thử-sai. |
| 8 | **Đóng góp cho dự án** | x | | | Tự làm InsightShare end-to-end và bổ sung vài tính năng thêm (xóa file, link có hạn, giới hạn upload). |

### Cần cải thiện

* **Kỷ luật:** chấp hành nội quy về giờ giấc và trang phục.
* **Tư duy giải quyết vấn đề:** phân tích vấn đề có hệ thống thay vì thử-sai.
* **Giao tiếp:** trao đổi trực tiếp và đặt câu hỏi nhiều hơn.
